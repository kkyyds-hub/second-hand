# FrontDay03 联调准备与验收

- 日期：`2026-03-12`
- 文档版本：`v1.0`
- 当前状态：`已完成并回填`

---

## 1. 环境前置条件

1. 后端服务可访问，默认以 `http://localhost:8080` 为准。
2. 前端以 `demo-admin-ui` 项目运行，常用命令为 `npm run dev:real`。
3. 当前页面需要的管理员账号与 token 可用。

---

## 2. 当日联调清单

1. 商品审核列表可进入且不白屏。
2. 点击通过动作后能正常调用接口。
3. 点击驳回后必须输入原因并提交。
4. 系统设置页可从菜单进入且显示占位结构。

---

## 3. 当日需要保留的证据

1. 商品审核列表请求记录。
2. 通过/驳回动作请求与页面反馈记录。
3. 字段缺失时页面兜底展示记录。

---

## 4. 验收判定建议

1. 能实际执行的项，优先记录为“运行态已确认”。
2. 只有代码存在但未实测的项，记录为“代码已完成待运行验证”。
3. 联调失败时，必须注明是前端问题、后端问题还是环境阻塞。
