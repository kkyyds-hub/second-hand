1. 后端服务可访问，默认以 `http://localhost:8080` 为准。
2. 前端以 `demo-admin-ui` 项目运行，常用命令为 `npm run dev:real`。
3. 当前页面需要的管理员账号与 token 可用。

## 4. 验收判定建议
