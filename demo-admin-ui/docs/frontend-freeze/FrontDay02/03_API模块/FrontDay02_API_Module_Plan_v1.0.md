## 3. 当日 API 规则

1. 用户管理页相关字段映射应留在 `src/api/user.ts` 或紧邻页面的逻辑中。
2. 封禁理由必须带入接口，不能只留在前端展示。
3. 成功后列表刷新策略要保持一致，避免不同动作行为分裂。
