## 2. 为什么 Day03 接在 Day02 之后

1. 市场域依赖 Day01 的鉴权壳与 Day02 之后更稳定的账户基线，但不应该和账户域混写。
2. 后端已存在 `MarketProductController`、`FavoriteController`、`ReviewController`，而用户端前端尚无对应页面与 API 模块。
3. 若不先建立 Day03 入口，商品列表、详情、收藏、评论、举报极易以“临时页面”方式零散推进，难以回填。
