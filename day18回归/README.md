## 4. 子目录职责

1. `事务治理`：事务边界、幂等、重试与恢复。  
2. `异步一致性治理`：Outbox 与消费幂等一致性。  
3. `缓存一致性治理`：缓存键模型、TTL 与一致性策略。  
4. `并发控制治理`：CAS 语义与并发分流口径。  
5. `安全治理`：敏感数据、输入安全、CSRF、安全扫描。  
6. `审计与监控治理`：审计日志、告警规则、日志平台接入。  
7. `风控治理`：信用规则、登录风控、异常交易规则。  
8. `性能治理`：索引回归、限流削峰、缓存性能。  
9. `执行复现步骤`：每个 Step 的复现步骤。  
10. `执行记录`：动态验证结果、报告与回填记录。  
11. `文档冻结与移交`：Day18 最终冻结与移交入口。
