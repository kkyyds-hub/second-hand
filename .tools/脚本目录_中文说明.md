# .tools 脚本目录（中文说明）

## 1. 为什么有中英文两个文件名

1. 中文脚本名是主入口，方便直接看懂用途。  
2. 英文脚本名是兼容入口，只做转发，避免历史命令失效。  

## 2. 中文主脚本（推荐直接使用）

1. `Day19_P1_S1_观测入口与口径校验.ps1`  
2. `Day18_闭环回归验证.ps1`  
3. `Day18_P5_S2_审计闭环校验.ps1`  
4. `密钥泄漏扫描.ps1`  

## 3. 旧英文名与新中文名映射

1. `day19-p1-s1-observe.ps1` -> `Day19_P1_S1_观测入口与口径校验.ps1`  
2. `day18-close-loop.ps1` -> `Day18_闭环回归验证.ps1`  
3. `day18-p5s2-audit-close-loop.ps1` -> `Day18_P5_S2_审计闭环校验.ps1`  
4. `secret-scan.ps1` -> `密钥泄漏扫描.ps1`  

## 4. 建议

1. 新文档、新命令统一写中文脚本名。  
2. 兼容入口保留一段时间，后续可按需删除。
